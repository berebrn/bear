******************************************************
一个逗音小插件, 功能：

<复制链接> 短按：复制视频或者图片链接; 长按：复制文案;

<视频下载> 下载当前无水印视频或者图片;

<音频下载> 下载当前播放视频的音频;

<UI设置> 

  --移除主页发布加号图标
  
  --状态栏透明
  
  --状态栏隐藏
  
  --主页顶栏透明化

  --评论区色调跟随主题
  
  --主页添加自动播放按钮
  
  --主页标签隐藏设置

<其他设置> 
  
  --移除开屏/主页的AD
  
  --禁止青少年提示框
  
  --禁止版本更新弹框
  
  --自动播放下一个视频

  --重新适配
  
<关于模块> 
  
  --模块信息

******************************************************
使用说明：

1、首次或者更新版本后会弹出适配提示框，待适配完成重启生效；

2、视频播放界面, 点击分享按钮弹出功能菜单；

3、理论上可以自动适配24以上的所有版本;
